# Realm of The Mad Satan
<hr>

## How to install

1. Clone repository
```sh
git clone https://github.com/N3nT/Realm-of-the-Mad-God
```

2. Install dependencies
```sh
pip install -r requirements.txt
```

3. Run script

```sh
python main.py
```

## Requirements
- [*pygame*](https://asd.pl) - free python game engine
